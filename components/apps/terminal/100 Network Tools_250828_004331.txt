Linux Networking Tools
1. ifconfig
Description: Displays information about network interfaces on the system, including IP addresses, MAC addresses, and more.

Example: ifconfig

2. ip
Description: A powerful command for displaying and configuring network interfaces. It provides more features than ifconfig.

Example: ip address show

3. ping
Description: Tests the reachability of a host on an IP network using ICMP echo requests.

Example: ping google.com

4. traceroute (tracepath)
Description: Displays the route that packets take to reach a destination, showing each hop along the way.

Example: traceroute google.com

5. netstat
Description: Displays information about network connections, routing tables, interface statistics, and more.

Example: netstat -a

6. ss
Description: A replacement for netstat. Provides information about sockets, network connections, and routing tables.

Example: ss -a

7. nslookup (dig)
Description: Performs DNS queries to translate domain names to IP addresses and vice versa.

Example: nslookup google.com

8. host
Description: Another tool for performing DNS queries, similar to nslookup.

Example: host google.com

9. route
Description: Displays and manipulates the IP routing table.

Example: route -n

10. iwconfig
Description: Configures wireless network interfaces and provides information about them.

Example: iwconfig

11. iw
Description: Similar to iwconfig, but part of the newer iw toolset for wireless configuration.

Example: iw dev wlan0 link

12. nmap
Description: A powerful and flexible network scanning tool used for discovering hosts and services on a computer network.

Example: nmap -sP 192.168.1.0/24

13. tcpdump
Description: A packet analyzer that allows the user to display TCP, UDP, and other packets being transmitted or received on a network.

Example: sudo tcpdump -i eth0

14. wget
Description: A command-line utility for retrieving files from the web. Useful for testing network connectivity and downloading files.

Example: wget 

15. curl
Description: Another command-line tool for transferring data with URLs. It supports a wide range of protocols and is often used for making HTTP requests.

Example: curl 

16. ssh
Description: Secure Shell (SSH) is a protocol for securely accessing a remote machine. It provides encrypted communication over a network.

Example: ssh user@hostname

17. scp
Description: Stands for Secure Copy. It allows secure file transfers between a local host and a remote host or between two remote hosts.

Example: scp file.txt user@remote:/path

18. iptraf
Description: An interactive and colorful IP LAN monitor that provides information about IP traffic, including statistics on TCP, UDP, ICMP, and more.

Example: sudo iptraf

19. iftop
Description: A real-time console-based network bandwidth monitoring tool. It shows a list of network connections and their data rates.

Example: sudo iftop

20. iperf
Description: Measures the maximum TCP and UDP bandwidth performance. It's often used to test network performance between two systems.

Example (server): iperf -s

Example (client): iperf -c server_ip
